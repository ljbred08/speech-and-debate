## File Structure

Each evidence card folder contains:

- **Evidence Cards (.md files)** - Structured debate evidence with tag lines, verbal citations, context, and impact analysis
- **Source Documentation (-source.md files)** - Complete original source content for verification
- **PDF Files** - Formatted versions ready for tournament printing and evidence exchange

## Evidence Card Format

All evidence cards follow a standardized table format with these columns:

- **Tag Line** - Concise summary of main argument
- **Verbal Citation** - Complete quotes with source attribution for debate rounds
- **What the evidence says in context** - Full explanation of source arguments
- **Impact** - Analysis connecting evidence to debate positions
- **Bibliographic Information** - Complete citation details (title, URL, author, dates)

## Quality Standards

All evidence cards meet rigorous standards:

- **Original Sources Only** - No encyclopedia excerpts or secondary summaries
- **Authoritative Sources** - Court decisions, government documents, academic papers, reputable think tanks
- **Complete Verification** - All quotes verified word-for-word from original sources
- **Proper Attribution** - Full context including speaker credentials and institutional authority
- **Substantial Content** - Complete, meaningful sentences for effective debate use

## Usage Guidelines

### For Tournament Competition

1. **Use PDF Files** - Print PDF versions for debate rounds and evidence exchange

### For Research and Verification
